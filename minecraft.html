Rules

This is a private server, so it's mostly based on trust. However, there are some obvious and some less obvious rules to follow. Not doing so results in a removal from the whitelist.

  • Live and let live
    1. No griefing
    2. No stealing
    3. Just ask before taking anything
  • Height limit of 128. Projects requiring more will need permission.

Commands

mail Checks your mailbox
w username message Sends mail to the given user, even if offline
m username message Sends a PM to the given user
sethome Sets your home
home Teleport to your home
tpa username Teleport to another user
nick Changes your own nickname
realname nickname See who is hiding behind that atrocious nickname-
recipe item See the crafting recipe for the given item
